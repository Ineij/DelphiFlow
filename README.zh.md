## ✨ 项目简介

**DelphiFlow** 是我的第一个 Vibe Coding 作品 —— 一个专为**德尔菲法问卷调研**设计的网页平台，面向学生与学术研究者。

问卷设计者与专家受访者通过**同一个入口**完成各自的工作，无需管理多个工具或链接，最大程度简化使用流程。

---

## 🚀 功能特色

- 📝 **自由编辑问卷** —— 自由添加题目、嵌入图片、拖拽排序、配置评分等级
- 👥 **统一入口** —— 问卷设计者与专家受访者共用同一页面，降低使用门槛
- 🔄 **专为德尔菲设计** —— 针对多轮德尔菲问卷调研流程深度优化

## 🛠️ 自行部署

由于本平台为个人托管，**建议有需要的用户 Fork 本仓库并自行部署**：

1. **Fork** 本仓库
2. 在 [supabase.com](https://supabase.com) 创建免费项目
3. 在 `lib` 目录中填入你的 Supabase 配置信息
4. 一键部署至 [Vercel](https://vercel.com)

```
NEXT_PUBLIC_SUPABASE_URL=你的_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=你的_supabase_anon_key
```

> 💡 部署完成后，你将拥有完全独立的实例，包含专属数据库与域名。
